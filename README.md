创建一个结构清晰、内容丰富的 GitHub 开源项目，汇总网上所有关于 AI Agent 开发的面试经验和常见面试题，帮助求职者系统性地准备 Agent 方向的面试。

## 🌟 项目简介

随着大语言模型（LLM）的发展，AI Agent 成为了最热门的技术方向之一。本项目旨在为准备 AI Agent / LLM 算法工程师、开发工程师的求职者提供一份**系统、全面、高质量的面经与面试题库**。

项目内容基于公开网络资源整理汇总（包括 CSDN、掘金、知乎、牛客网及国外优质技术社区），涵盖了从 LLM 基础到 Agent 架构设计的全方位知识体系。
