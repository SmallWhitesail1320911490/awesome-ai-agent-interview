## 高频面试题 

### 1. 什么是 AI Agent？它包含哪些核心组件？
**参考答案**：
AI Agent（人工智能代理）是一个可以通过感知环境并采取行动以实现特定目标的自主系统。在基于大周型的上下文中，Agent 通常以 LLM 作为"大脑"。
核心组件通常包括：
1. **大脑 (Brain / LLM)**：负责理解、推理、规划和决策。
2. **记忆 (Memory)**：分为短期记忆（上下文历史）和长期记忆（外部存储如向量数据库）。
3. **规划 (Planning)**：包含任务分解（Task Decomposition）和自我反思（Self-Reflection）。
4. **工具 (Tools/Action)**：赋予 Agent 与外界交互的能力，如搜索网络、读写文件、执行代码等。

### 2. 请说出 Agent 和纯大语言模型（如 ChatGPT 初期版本）的主要区别？
**参考答案**：
纯 LLM 通常被动地回答问题，主要依赖训练数据中的静态知识，缺乏主动获取新信息或改变外部状态的能力。
Agent 具有：
- **主动性**：可以根据目标自我规划多步操作。
- **环境交互**：可以通过工具调用感知和改变外部世界。
- **持久记忆**：可以结合记忆机制维持超越单次会话的上下文理解。
