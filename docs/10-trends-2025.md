## 高频面试题

### 1. 什么是多模态 Agent？它比纯文本 Agent 多了哪些能力？
**参考答案**：
多模态 Agent 不仅能理解和生成文本，还能同时处理图像、音频、视频等其他模态的数据（如 GPT-4o 或 Gemini 1.5 Pro 原生多模态模型驱动的代理）。
额外能力：
- 能够直接"看屏"（例如视觉理解网页，进行 UI 自动化测试操作）。
- 能够直接听取人类语音指令，根据环境背景音做出反应。
- 在具身智能领域能够处理摄像头的实时视频流，执行复杂的物理动作规划。
