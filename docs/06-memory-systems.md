## 高频面试题

### 1. Agent 系统中短期记忆和长期记忆分别是如何实现的？
**参考答案**：
- **短期记忆**：通常指当前 Session（会话）的消息历史。实现方式是直接保存在上下文窗口中（或利用滑动窗口、摘要机制防止超出 Token 限制）。
- **长期记忆**：指跨越会话持久存储的信息（如用户偏好、过往历史总结）。实现方式通常是将信息 embedding 并存入向量数据库，或结构化存入关系型/图数据库，在需要时进行 RAG 召回。
