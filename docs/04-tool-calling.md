## 高频面试题

### 1. Function Calling 的原理是什么大模型在其中扮什么角色？
**参考答案**：
Function Calling 允许开发者向大模型描述一组函数的签名（名称、参数、功能），大模型会在需要时**决定调用哪个函数并生成调用所需的参数**（通常返回 JSON 格式）。
大模型不实际执行函数，而是完成意图识别和参数提取。开发者接收到 JSON 后在本地执行真实函数，然后将结果再喂给大模型以生成最终回答。

### 2. 什么是 MCP（Model Context Protocol）？它解决什么问题？
**参考答案**：
MCP 是一项开放标准，旨在大语言模型与外部数据源或工具之间提供标准化的双向通信协议。它解决了过去各个模型和工具集成标准不统一、开发成本高的问题。通过 MCP，Agent 可以以标准一致的方式查询各类服务、数据库和执行指令。
