## 高频面试题

### 1. 为什么要使用多 Agent 系统？相比单 Agent 有什么优势？
**参考答案**：
多 Agent 系统通过让多个具有各自“人设”（角色）的 Agent 协同工作来解决更复杂的问题。
优势：
1. **专业化分工**：每个 Agent 专注于自己擅长的领域（如一个写代码，一个做测试）。
2. **减少幻觉和偏见**：不同角色的 Agent 可以互相 Review 和辩论。
3. **更高的成功率**：拆分复杂流程，使得上下文更干净聚焦。
