## 高频面试题

### 1. 设计一个自动代码评审（Code Review）Agent，你会如何设计它的工作流？
**参考答案**：
1. **触发与获取**：监听 Git Webhook（例如 PR 提交），获取变更的 Diff 文件。
2. **任务分解**：Diff 较大时，使用一个 Planning Agent 将 Diff 按文件/模块切分开来。
3. **多维审核**：使用多个并行的 Agent 角色进行审核（如：安全专家检查漏洞，性能专家检查复杂度，业务专家检查逻辑一致性）。
4. **汇总与反馈**：总结专家 Agent 的意见，格式化输出 markdown 评论，调用 GitHub API 回写到 PR 中。
