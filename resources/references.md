## 一、必读论文

| 论文 | 年份 | 核心贡献 |
|------|------|----------|
| [Attention Is All You Need](https://arxiv.org/abs/1706.03762) | 2017 | 提出 Transformer 架构 |
| [ReAct: Synergizing Reasoning and Acting](https://arxiv.org/abs/2210.03629) | 2022 | Agent 推理与行动框架 |
| [Chain-of-Thought Prompting](https://arxiv.org/abs/2201.11903) | 2022 | 思维链推理 |
| [Tree of Thoughts](https://arxiv.org/abs/2305.10601) | 2023 | 树状推理探索 |
| [Generative Agents: Interactive Simulacra](https://arxiv.org/abs/2304.03442) | 2023 | 斯坦福小镇，Agent 记忆系统 |
| [Toolformer](https://arxiv.org/abs/2302.04761) | 2023 | 模型自学使用工具 |
| [Self-Refine](https://arxiv.org/abs/2303.17651) | 2023 | 自我迭代优化 |
| [Reflexion](https://arxiv.org/abs/2303.11366) | 2023 | 语言 Agent 的自我反思 |
| [HuggingGPT](https://arxiv.org/abs/2303.17580) | 2023 | LLM 调度多模型协作 |
| [Self-RAG](https://arxiv.org/abs/2310.11511) | 2023 | 自适应检索增强生成 |
| [LoRA: Low-Rank Adaptation](https://arxiv.org/abs/2106.09685) | 2021 | 参数高效微调 |
| [Flash Attention](https://arxiv.org/abs/2205.14135) | 2022 | IO-aware 高效注意力 |
| [Efficient Memory Management (PagedAttention)](https://arxiv.org/abs/2309.06180) | 2023 | vLLM 核心优化 |
